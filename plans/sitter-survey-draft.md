| Interview Theme | Frequency | Survey Questions |
|---|---|---|
| Fees too high / take clients off-platform | 6/7 | Q9, Q10, Q11 |
| Meet & Greet is essential | 5/7 | Q12, Q28 |
| Insurance / sitter protection | 5/7 | Q13, Q15 |
| Reviews are critical but inflated | 4/7 | Q16, Q17, Q18, Q19 |
| Expense tracking for taxes | 4/7 | Q20, Q21, Q22 |
| CRM / client management | 3/7 | Q20 |
| Bad UX / buggy apps | 3/7 | Q8 |
| Hard to get discovered as new sitter | 3/7 | Q8, Q25 |
| Analytics / profile insights | 3/7 | Q20 |

### PetLink features already built (for context when analyzing responses)

- Emergency contacts (owner + sitter)
- Species-aware profiles (dog, cat, bird, reptile, small animal)
- Walk tracking with GPS + photo/video
- Care task checklists from pet care instructions
- Sub-rating reviews with blind window
- Analytics (revenue, clients, profile views)
- Booking calendar + availability management
- Sitter portfolio posts (Instagram-style)
- Promote page with QR code
- Verification / background checks
- Notification preferences
- Cancellation policies
- Real-time messaging via Socket.io
- Pro subscription tier
